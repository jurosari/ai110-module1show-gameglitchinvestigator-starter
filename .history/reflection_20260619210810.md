## 4. What did you learn about Streamlit and state?

- How would you explain Streamlit "reruns" and session state to a friend who has never used Streamlit?
* Every time you interact with a Streamlit app, Streamlit re-runs your entire script from top to bottom, so any normal variable gets recreated from scratch each time. Session state is a special dictionary that survives those reruns, so you store anything that needs to persist in st.session_state instead of in a regular variable.
